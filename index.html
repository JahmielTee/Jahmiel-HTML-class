7 Days to HTML Mastery

Step-by-step learning path designed by Jahmiel

Day 1 – HTML Skeleton & Structure

Foundation

Concept Snapshot

Every HTML page starts with <!DOCTYPE html>. Use <html>, <head>, <title>, <body> to build the skeleton.

Code Example

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is my first paragraph!</p>
</body>
</html>

Mini Challenges

  • Change the title inside <title> to your name
  • Add a second <p> paragraph with a short description of yourself
Pro Tip

Always include the <!DOCTYPE html> declaration - it tells the browser you're using HTML5!

Day 2 – Headings & Paragraphs

Text Basics

Concept Snapshot

<h1> to <h6> are headings (H1 is biggest). <p> is for paragraphs.

Code Example

<h1>My Biography</h1>
<h2>About Me</h2>
<p>I love learning web development.</p>
<p>My dream is to build my own website soon.</p>

Mini Challenges

  • Create 3 headings (<h1>, <h2>, <h3>) with different titles
  • Write 3 short paragraphs under them
Remember

Use only one <h1> per page for SEO best practices!

Day 3 – Links & Images

Multimedia

Concept Snapshot

<a href='URL'> creates a link. <img src='image.jpg' alt='description'> displays an image.

Code Example

<a href="https://google.com" target="_blank">Visit Google</a>
<img src="myphoto.jpg" alt="Me smiling" width="200">

Mini Challenges

  • Add a link to your favorite website (make it open in a new tab)
  • Add any image (download one and put it in the same folder)

Day 4 – Lists

Organization

Concept Snapshot

<ul> = unordered list, <ol> = ordered list, <li> = list item.

Code Example

<h2>My Grocery List</h2>
<ul>
    <li>Bread</li>
    <li>Milk</li>
    <li>Eggs</li>
</ul>

<h2>Steps to Get Ready</h2>
<ol>
    <li>Wake up</li>
    <li>Brush teeth</li>
    <li>Eat breakfast</li>
</ol>

Mini Challenges

  • Make a list of your top 5 favorite songs, foods, or hobbies
  • Try both <ul> and <ol> and see the difference
Quick Tip

Use <ul> for navigation menus and <ol> for step-by-step instructions!

Day 5 – Divs, Spans & Line Breaks

Structure

Concept Snapshot

<div> groups content, <span> styles inline text, <br> makes a line break.

Code Example

<div>
    <h2>My Section</h2>
    <p>This is one section of my page.</p>
</div>

<p>This is normal text but
<span style="color: red;">this part is red!</span></p>

<p>Line one<br>Line two<br>Line three</p>

Mini Challenges

  • Create a section <div> with a heading and paragraph inside
  • Use <span> to color one word differently
Did You Know?

<div> is a block-level element while <span> is inline!

Day 6 – Forms (Intro)

Interactivity

Concept Snapshot

<form> groups input fields. <input>, <textarea>, <button> let users type or click.

Code Example

<form>
    <label>Name:</label><br>
    <input type="text"><br><br>
    <label>Email:</label><br>
    <input type="email"><br><br>
    <label>Message:</label><br>
    <textarea></textarea><br><br>
    <button type="submit">Send</button>
</form>

Mini Challenges

  • Add a new input for 'Age' (<input type='number'>)
  • Add a checkbox with the label 'I agree'
Pro Tip

Always use proper <label> elements for accessibility!

Day 7 – Final Project

Capstone

Concept Snapshot

Combine everything into a single mini-site.

Code Example

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>Hello! I'm learning HTML and this is my first site.</p>
    <h2>My Hobbies</h2>
    <ul>
        <li>Reading</li>
        <li>Coding</li>
        <li>Music</li>
    </ul>
    <h2>Favorite Link</h2>
    <a href="https://example.com" target="_blank">Check this out</a>
    <h2>Contact Me</h2>
    <form>
        <label>Name:</label><br>
        <input type="text"><br><br>
        <button type="submit">Send</button>
    </form>
</body>
</html>

Mini Challenges

  • Add an image of yourself or something you like
  • Add another section with <h2> and a paragraph
  • Make it look neat and presentable — this is your first real website!
Congratulations!

You've completed the 7-day HTML bootcamp! You now have the foundation to build any website.
